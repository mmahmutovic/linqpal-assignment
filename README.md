# linqpal-assignment
Technical assignment in MERN stack for Linqpal
## Available Scripts

Install all packages and seed the db:
```
sh install-app.sh
```

Start server and app:

```
sh start-app.sh
```

And kill processes by using script:
```
sh stop-app.sh
```
